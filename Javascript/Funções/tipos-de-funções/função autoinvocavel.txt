IIFE(Immediately Invoked Function Expression)

Uma função anónima entre parênteses, seguida por outra par de pareênteses, que representa sua chamada;

(
  function() {
    let name + "Digital Innovation One"
    return name;
 }
)();

Também pode ser utilizada com parametros ou armanezada em uma variavel;

( 
  function(a,b) {
	return a + b;
}
)(1,2);

ou 

const soma = 3 (
   function() {
	return a + b;
  }
)(1, 2);
