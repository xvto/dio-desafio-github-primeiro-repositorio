Podemos usar palavras reservadas "if" para estabelcer uma condição:

Ex:  var jogador 1 = 0;
     var jogador 2 = 0;
      
     if (jogador1 > 0) {
         console.log('jogador1 marcou um ponto!')
         }

// como let: se o jogador 1 tiver valor maior que 0 ele marcou 1 ponto;
